SignEdit
SignEdit is used to detect the position of the sign in a sequence of images of video footage and integrate the tested sign image onto each image. This is release version for Windown x64 operation system. 

Installation
Opencv is needed to use this programe. 
Opencv should be downloaded and loaded on C:.
After opencv is installed, opencv_world320.dll file still can be missing. If error message of opencv_world320 missing shows up, the missing .dll file need to be downloaded and loaded in system folder (usually C:\Windows\System32)

Usage
Double click ImgPrc.exe.
Open the images (O)
Detect the Edge.
Insert new sign image(2).

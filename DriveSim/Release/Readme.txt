DriveSim
It is used to play video and motion on simulator. 

Installation
Opencv need to be installed on C:.

Usage
Put DriveSim.exe in the same directory with the folders containing the images of the video. A txt file specifying the order of the trials is also needed. 
Double click DriveSim.exe to run.
Use buttons to begin and end each trial. The buttons need to be defined before used.
